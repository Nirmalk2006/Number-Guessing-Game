* JavaScriptut Projects

This repository contains small JavaScript mini-projects and practice examples.

* No.Guessing (Number Guessing Game)

Guess a random number between 1 and 100.

*Files
- `NoGuessing/index.html`
- `NoGuessing/Script.js`
- `NoGuessing/nogue.css`

* Run
1. Open `NoGuessing/index.html` in a browser.
2. Enter a guess and click **Submit Guess**.
3. After the game ends, use **Start New Game** to play again.
